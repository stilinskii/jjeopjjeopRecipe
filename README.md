### 인덱스, 통합검색

```
처음 방문하면 나오는 페이지. 재료나 레시피 이름을 통해 레시피와 상품 통합검색으로 넘어갑니다. 페이지에는 스크랩 순, 판매량 순으로 정렬된 레시피와 상품을 볼 수 있고 더보기를 통해 각각 페이지에 들어갈 수 있습니다.
```
